## Conjugación
### Presente
| singular             | plural                      |
|----------------------|-----------------------------|
| yo debo              | nosotros debemos            |
| tú debes             | vosotros debéis             |
| él, ella, usted debe | ellos, ellas, ustedes deben |
| vos te debés         |                             |
      
### Ejemplo
Osvaldo y Pedro me deben mucho dinero.
