# Romper

To break  
![to break](img/romper.webp)

## Conjugación

### Presente

| singular              | plural                       |
|-----------------------|------------------------------|
| yo rompo              | nosotros rompemos            |
| tú rompes             | vosotros rompéis             |
| él, ella, usted rompe | ellos, ellas, ustedes rompen |
| vos te rompés         |                              |

### Ejemplo

Los algoritmos crean patrones, no los rompen.
