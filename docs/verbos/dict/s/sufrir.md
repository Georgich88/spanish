# Sufrir

To suffer  
![To suffer](img/sufrir.webp)

## Conjugación

### Presente

| singular              | plural                       |
|-----------------------|------------------------------|
| yo sufro              | nosotros sufrimos            |
| tú sufres             | vosotros sufrís              |
| él, ella, usted sufre | ellos, ellas, ustedes sufren |
| vos te sufrís         |                              |

### Ejemplo

Ahora todos debemos sufrir a un país que nos odia.
