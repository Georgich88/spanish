# Haber

to have (to do something, auxiliary verb)

## Conjugación

### Presente

| singular           | plural                    |
|--------------------|---------------------------|
| yo he              | nosotros hemos            |
| tú has             | vosotros habéis           |
| él, ella, usted ha | ellos, ellas, ustedes han |
| vos has            |                           |
