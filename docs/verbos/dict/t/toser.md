# Toser

To cough

## Conjugación

### Presente

| singular             | plural                      |
|----------------------|-----------------------------|
| yo toso              | nosotros tosemos            |
| tú toses             | vosotros toséis             |
| él, ella, usted tose | ellos, ellas, ustedes tosen |
| vos te tosés         |                             |

### Ejemplo

Si tiene que toser sobre alguien, encuentre un sirviente.
