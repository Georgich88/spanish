# Subir

To go up  
![To go up](img/subir.png)

## Conjugación

### Presente

| singular             | plural                      |
|----------------------|-----------------------------|
| yo subo              | nosotros subimos            |
| tú subes             | vosotros subís              |
| él, ella, usted sube | ellos, ellas, ustedes suben |
| vos te subís         |                             |

### Ejemplo

Los precios suben cada dia.
